# Instagram DM automation

The source code repository for Instagram DM automation.

## Getting Started

### Install

```bash
% git clone git@github.com:keithboice/example_instagram_messages.git

% cd example_instagram_messsages

% npm install
```

### Develop

```bash
% npm run dev
```

### Test

```bash
% npm run test
```

### Production

```bash
% npm run start
```
